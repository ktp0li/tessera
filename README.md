# Tessera [![tessera_keepass_bot.t.me](https://img.shields.io/badge/Telegram-Bot-blue)](https://tessera_keepass_bot.t.me)

### Usage:
1. Paste token into TOKEN env var in docker-compose.yml
2. `docker-compose build`
3. `docker-compose up`
